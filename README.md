# Умная сортировка файлов

Скрипт для автоматической сортировки файлов в выбранной папке по расширению или по дате.

## Возможности

- **Выбор папки** — диалог выбора или ввод пути вручную
- **Сортировка по расширению** — создаются папки jpg, png, mp4, pdf и т.д.
- **Сортировка по дате** — папки по дате изменения (YYYY-MM-DD)
- **Автоматическое создание папок** — подпапки создаются по мере необходимости
- **Лог в консоль** — отображается, какой файл куда перенесён

## Требования

- Python 3.6+
- Стандартная библиотека (дополнительные пакеты не нужны)

## Установка и запуск

```bash
python smart_sort.py
```

## Использование

1. Запустите скрипт
2. Выберите папку в диалоговом окне (или введите путь)
3. Выберите режим сортировки:
   - **1** — по расширению
   - **2** — по дате
4. Файлы будут перемещены, в консоли появится лог переносов

## Пример вывода

```
==================================================
  Умная сортировка файлов
==================================================

Папка: C:\Users\...\Downloads

Режим сортировки:
  1 - По расширению (jpg, png, mp4, pdf и т.д.)
  2 - По дате создания
Выберите (1 или 2): 1

--- Лог переносов ---

  photo.jpg -> jpg/
  image.png -> png/
  document.pdf -> pdf/

--- Готово. Перенесено файлов: 3 ---
```
